# Expense-keEpr

Manage your daily expenses with this basic app created using react, Backend is running on ExpressJS and MongoDB as database server.

## Current Features:
- Add expense
- View expenses in Summary section
- Update any expense
- Delete an expense

## Sub-Planned Features:
- Pagination
- Brief expense summary (with or without filters)

## Roadmap Features:
- Advanced filter to get desired data.
- Graphs & Charts for more insights and analysis.
- Bulk upload via CSV file.
- Data download in Excel/CSV format.
